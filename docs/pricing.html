常见问题

什么是 Token？
Token 是 AI 模型处理文本的基本单位。一个汉字约等于 1-2 个 Token，50 万 Token 大约可以处理 25-50 万字的文本。
Token 用完了怎么办？
Token 额度每月初自动重置。如果当月用完，可以等待下月重置或升级套餐。
支持哪些支付方式？
支持微信支付和支付宝，方便快捷。
可以随时取消订阅吗？
可以。取消后，当前周期内仍可继续使用，到期后不再自动续费。
